# SATO Luxury IP Generator - Starfield Edition

## English

### Overview

SATO Luxury IP Generator is a premium interactive PHP web application
designed with a modern glassmorphism interface, starfield animation, and
a dynamic IP generation experience.

The project uses a lightweight architecture based on: - PHP - Vanilla
JavaScript - HTML5 - CSS3 - Interactive UI effects

### Features

-   Premium glassmorphism design
-   Animated starfield background
-   Interactive region selection
-   Dynamic IP generator interface
-   Copy generated IP functionality
-   Responsive mobile-first design
-   No database required
-   No external frameworks required

### Requirements

Before installation, make sure you have:

-   PHP 7.4 or newer
-   Apache / Nginx web server
-   A modern web browser

### Installation

1.  Download the project files.
2.  Place the PHP file inside your web server directory.

Example:

``` bash
/var/www/html/
```

3.  Start your web server.

For PHP built-in server:

``` bash
php -S localhost:8000
```

4.  Open your browser:

``` text
http://localhost:8000
```

### Required Libraries Installation

This project does not require external libraries.

No Composer installation is needed:

``` bash
# No dependencies required
```

### Usage

1.  Open the website.
2.  Select your preferred region.
3.  Click the generate button.
4.  Copy the generated result.
